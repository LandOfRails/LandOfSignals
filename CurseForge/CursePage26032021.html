LandOfSignals

 

Currently accepted Minecraft versions: 1.7.10, 1.10.2 - 1.16.5

 

Requires Universal Mod Core

 

Discord: https://discord.gg/ykAqHKYjVM

 

This mod adds many different railroad signals, as well as railroad assets from different countries.

 

Compatibility

 

The mod is compatible with all existing railroad mods. This is made possible by addressing the signals with a universal box. When this gets a redstone signal the displayed signal image changes. In addition, you can determine the signal height independently, because you assemble the signals themselves / put on top of each other. Thus the height can be adapted to e.g. Traincraft, but also to Immersive Railroading.

 

Usage

 

Use the connector to bind a signal with the signal box.

Custom content

 

You can add your own signals and create content packs that others can use.

Learn how to create your own here: Wiki
